# 덕이킨더바움 행복부자 통장

덕이킨더바움의 행복숲, 행복통장, 성장기록, 미션 기능을 모바일 화면에 맞춘
역할별 웹앱입니다.

## 주요 기능

- **원장용 화면**: 모든 반, 모든 아이의 행복통장/행복숲/성장기록/미션 조회
- **교사용 화면**: 자기 반 아이만 조회, 자기 반 대상 미션 생성 및 완료 처리
- **학부모용 화면**: 자기 자녀의 행복통장, 성장기록, 미션 수행내역만 조회
- **미션 자동 관리**
  - 앱 실행 중 매일 오전 0시가 지나면 반복 미션이 새 날짜 미션으로 자동 생성됩니다.
  - 완료한 미션은 이전 날짜 기록으로 유지되고, 다음날 미션 완료 상태는 자동으로 초기화됩니다.
  - 교사는 미션 생성 시 매일 반복 여부를 선택할 수 있습니다.
- **모바일 최적화**: 카드형 화면, 하단 탭 내비게이션, 터치 친화 버튼을 적용했습니다.

## 실행

```bash
npm start
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 검증

```bash
npm test
npm run build
```

## 배포 파일 생성

```bash
npm run build
```

정적 파일이 `dist/`에 생성됩니다.
